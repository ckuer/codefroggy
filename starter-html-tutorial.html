Starter HTML page structure guide
  • Tutorial
  • Extra Resources

It can be hard to keep track of the different elements to be found on an HTML page. Let's go over what a common HTML document structure looks like!

Above I included a screenshot of some basic HTML generated in Visual Studio Code (a free code editor by Microsoft used by both pros and beginners). Let's pick it apart element by element.

The !DOCTYPE Declaration

This is required for every HTML file. It tells the browser what it's dealing with.

The html tag and lang attribute

The html tag is, simply enough, the root to contain all your HTML code! Note that it requires a closing tag way at the end, as it wraps around the rest of the content.

The lang attribute specifies the language of the page's content, which in this case is "en" for English.

The head tag

Metadata about your page is written up here. There are many elements not shown that typically go here, such as links to external stylesheets.

The charset attribute in the first meta tag specifies the text character set your page is using.

The second meta tag with a viewport name attribute determines how your content displays in the viewport, or the screen/window being used to view your site. What's shown in the screenshot indicates the view is 1:1 with the user's device width.

These details may seem cloudy at the moment, but trust me that over time they'll grow much more familiar to you!

The title tag

This one is straightforward: it's the title of your page, and what displays in the browser tab up above. Never forget to fill this in!

The body tag

Finally, the body element. In here you'll write the code for what shows up on the page. I'm typing inside the body tag right now.

There is a wide selection of elements you can make use of inside the body, but every learner starts simple.

With learning to code, it's a gradual process with a lot of repeated learning! In the end the designs you come up with and solutions you'll find can be very rewarding.

Oh, and by the way...

A nifty VS Code shortcut for HTML files

Not wanting to type out all these elements manually?

Create and/or open an .html file in Visual Studio Code. If you have IntelliSense enabled, when you start typing "html" a menu will come up with some suggestions.

Select html:5 and an HTML 5 document will be generated for you. Use this trick whenever you need a quick start!
